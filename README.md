## Reminder: How to Stage & Push (in VS Code)
**0.** SAVE files  
**1.** ctrl+tilde to open Terminal  
**2.** Type: git add -A  
**3.** Press Enter  
**4.** Type: git commit -m "what you're committing" (message between quotes)  
**5.** Press Enter  
**6.** Type: git push  
**7.** Press Enter
